# Player Movesets

Every player should implement all of the following moves, though they can vary what they actually do.

| move | inputs | modifiers |
| - | - | - |
| move left | left |
| move right | right |
| crouch | down |
| jump | jump |
| punch | attack |
| punch up | up, attack |
| crouch punch | down, attack |
| kick | special |
| kick up | up, special |
| leg swipe | down, special |
